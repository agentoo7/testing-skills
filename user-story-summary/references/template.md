- **Title:** {{Story Title}}

- **Preconditions:**
{{Preconditions}}

  As a **{{User Role}}**,  
  I want **{{what the user wants}}**

  1. {{Acceptance Criterion 1}}  
  2. {{Acceptance Criterion 2}}  
  3. …

- **Related Stories:**  
  - {{Story ID / Title of related story}}

  - {{Any additional information or notes}}
