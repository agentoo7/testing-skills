# Functional Test Case Generator

Generate comprehensive, high-quality functional test cases from user story summaries following established templates and quality standards.

## When to Use This Skill

Use this skill when:
- User requests test case generation from user stories
- User provides a user story summary document
- User asks to create functional test cases or test scenarios
- User mentions test coverage creation for user stories

## Overview

This skill generates functional test cases through a structured workflow:

1. Verify input file (user story summary)
2. Identify place to store generated test cases
3. Gather context (project brief, templates, checklists)
4. Analyze user story relationships
5. Generate test cases per user story
6. Validate against quality checklist
7. Request user approval (per user story)
8. Generate final report
